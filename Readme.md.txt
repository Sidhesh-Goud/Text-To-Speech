# Text-To-Speech  A simple Text-To-Speech Converter built using HTML, CSS, and JavaScript. This project allows users to convert typed text into speech, utilizing the Web Speech API.  ## Live Demo  Check out the live demo [here](https://YourUsername.github.io/Text-To-Speech/).  ## Features  - Convert text to speech with multiple voice options - Clean and responsive UI - Simple to use and lightweight  ## How to Use  1. Type any text in the provided text area. 2. Select a voice from the dropdown menu. 3. Click on the "Listen" button to hear the text spoken aloud.  ## Technologies Used  - HTML - CSS - JavaScript - Web Speech API  ## License  This project is licensed under the MIT License - see the [LICENSE](LICENSE) file for details.
